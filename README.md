# 🕹️  SOBRE O PROJETO
 A PixelForge Studios é um estúdio independente fictício desenvolvido com propósito educacional.
Este projeto representa uma simulação do site oficial do estúdio, reunindo informações sobre seus jogos, trailer promocional, feedback da comunidade e um canal dedicado para contato.

A proposta é colocar em prática conhecimentos de HTML5, CSS e JavaScript na construção de uma experiência moderna, responsiva e imersiva, com identidade visual inspirada no universo dos games.
